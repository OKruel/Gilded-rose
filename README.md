# Gilded-rose

### RUN THE TESTS
1. Clone the github repository
2. Change directory into the server folder
3. Run __"yarn test"__ or __"npm test"__

### RUN THE APP
1. Clone the github repository
2. Change directory into the server folder
3. Run __"yarn install"__ or __"npm install"__
4. Run __"yarn dev"__ or __"npm run dev"__
